📘 Lecture 1 : HTML – Hyper Text Markup Language

HTML stands for Hyper Text Markup Language. It is the standard language used to create and structure webpages.

🔹 What is HTML?

  • HyperText: Ability to create links that connect web pages together.
  • Markup Language: Use of "tags" to give meaning and structure to plain text.

1️⃣ The Heading (<h1> to <h6>)

Used to define titles and subtitles, creating a logical hierarchy.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
  • <h1>: Main title, used once per page.
  • <h2>: Section heading.
  • <h3>: Sub-section heading.

2️⃣ The Paragraph (<p>)

Defines blocks of text. Browsers add spacing automatically before and after paragraphs.

<p>This is the first paragraph. It contains several sentences.</p>
<p>This is a second, separate paragraph.</p>

3️⃣ Horizontal Rule (<hr>)

Defines a thematic break, usually displayed as a horizontal line.

<p>This is the first topic.</p>
<hr />
<p>This is a completely different topic.</p>

4️⃣ Line Break (<br>)

Creates a single line break (like pressing Enter once).

<p>221B Baker Street<br>
London, England</p>

5️⃣ List Elements (<ul>, <ol>, <li>)

Used to create lists. Perfect for showing related items.

<ul>
  <li>Tea Bag</li>
  <li>Water</li>
  <li>Sugar</li>
  <li>Milk</li>
</ul>

<ol>
  <li>Boil water</li>
  <li>Add tea bag, sugar, and milk</li>
  <li>Keep boiling for 5 minutes</li>
  <li>Serve hot</li>
</ol>

6️⃣ The Anchor Tag (<a>)

Makes text or images clickable. Requires the href attribute.

<a href="https://www.coderarmy.in/" target="_blank">Visit Coder Army</a>

target attribute:

  • _self → Open in same window (default)
  • _blank → Open in new tab

7️⃣ The Image Tag (<img>)

Embeds images into a webpage. Requires src and alt attributes.

<img src="image.png" alt="Description of image">
